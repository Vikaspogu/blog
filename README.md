# Vikas Pogu Blog

## This is my personal blog

## Cloning repo

```
git clone https://github.com/Vikaspogu/blog.git
cd blog
rm -rf public
git submodule add -b master https://github.com/vikaspogu/vikaspogu.github.io.git public
hugo
```
